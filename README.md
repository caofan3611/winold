# winold

## 介绍
winold 是一个用于开发控制台游戏的库，旨在提供简单易用的接口，帮助开发者快速构建跨平台的控制台游戏应用。

> **注意：** 请勿下载带有 "CNU" 后缀的版本。带有 "CNU" 或 "dev" 后缀的版本可能会在正式版发布后被删除。

## 软件架构
该项目的目标是实现全平台兼容性，只要拥有合适的编译器，就可以在任何平台上使用。

## 安装教程
1. 下载对应版本的压缩包。
2. 解压后将文件放置在你的项目目录中。
3. 在代码中引用头文件：
   ```cpp
   #include "winold.h"
   ```
